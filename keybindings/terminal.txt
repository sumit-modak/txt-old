Ctrl+D: Send an end-of-file (EOF) character to the current command
Ctrl+K: Delete the text from the cursor position to the end of the line
Ctrl+U: Delete the text from the cursor position to the beginning of the line
Ctrl+W: Delete the word to the left of the cursor
Ctrl+Z: Suspend the current command and put it in the background
